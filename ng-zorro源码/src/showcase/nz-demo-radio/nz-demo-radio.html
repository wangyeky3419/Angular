Radio 单选框

单选框。

何时使用#

  • 用于在多个备选项中选中单个状态。

  • 和 Select 的区别是，Radio 所有选项默认可见，方便用户在比较中选择，因此选项不宜过多。

API

nz-radio-group

单选框组合，用于包裹一组 nz-radio/nz-radio-button。

参数 说明 类型 介绍 默认值
ngModel 用于设置当前选中的值 String 可以双向绑定 无
nzSize 大小，只对按钮样式生效 String 可选large default small default

[nz-radio]/[nz-radio-button](必须配合nz-radio-group使用)

参数 说明 类型 默认值
nzValue 当前Radio标示 String
nzDisabled 当前Radio是否不可选 Boolean false
