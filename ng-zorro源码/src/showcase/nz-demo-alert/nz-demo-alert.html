Alert 警告提示

警告提示，展现需要关注的信息。

何时使用

  • 对当某个页面需要向用户显示警告的信息时。

  • 非浮层的静态展现形式，始终展现，不会自动消失，用户可以点击关闭。

代码演示

最简单的用法。

显示关闭按钮，点击可关闭警告提示。

可口的图标让信息类型更加醒目。

共有四种样式 success、info、warning、error

含有辅助性文字介绍的警告提示。

可以自定义关闭，自定义的文字会替换原先的关闭 Icon

API

nz-alert

参数 说明 类型 默认值
nzType 必选参数，指定警告提示的样式，有四种选择 success、info、warning、error String info
nzCloseable 可选参数，默认不显示关闭按钮 Boolean false
nzCloseText 可选参数，自定义关闭按钮 String 无
nzMessage 与alert-body标签二选一，警告提示内容 String 无
nzDescription 可选参数，警告提示的辅助性文字介绍 String 无
nzOnClose 可选参数，关闭时触发的回调函数 function 无
nzShowIcon 可选参数，是否显示辅助图标 Boolean false
nzBanner 是否用作顶部公告 Boolean false
[alert-body] 与nzMessage二选一，定义Message ng-content 无
[alert-description] 可选参数，定义Description ng-content 无
